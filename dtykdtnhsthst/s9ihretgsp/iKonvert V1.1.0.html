iKonvert Modes

If you have set your iKonvert's DIP switches to any mode other than the default Mode 0, this application will not work. Only when the DIP switches are all OFF, will the iKonvert accept mode change commands from the app.

Please make sure your iKonvert's DIP switches are set to all OFF.

When you run this app, your iKonvert will be set to mode 15 and it will stay in this mode after you close the app.

If you wish to use the iKonvert in another mode, use the Select a mode function below to select the mode you wish to use. As long as you do not change the DIP switches or run this app again, your iKonvert will stay in the selected mode.
